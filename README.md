GitHubリポジトリをダウンロードし、すべてのファイルを同一フォルダ内に配置してください。

NI-MAXをインストールしていなければインストールしてください。

NI-MAXでKEITHLEY_2000_MULTIMETERとTektronix_TBS_1052B-EDUのドライバをインストールしてください。

TEXIO_FGX-2112のドライバは以下でインストールしてください。
https://www.texio.co.jp/product/detail/21

launcher.exeを起動してください。

launcher.exe は Microsoft Defender などのウイルス対策ソフトにより 誤検知される場合があります。 
その場合は launcher.py を実行してください。 実行には Pythonが必要です。
